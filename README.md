# 🧠 Kishore AI - Face Recognition & Emotion Detection System

An advanced AI-based real-time face recognition system with:

✔ Identity Verification  
✔ Eye Blink Detection (EAR Algorithm)  
✔ Emotion Recognition (DeepFace)  
✔ Real-Time Monitoring System  

## 🚀 Technologies Used
- Python
- OpenCV
- face_recognition
- DeepFace
- NumPy
- SciPy

## 🔍 Features
- Detects known & unknown faces
- Eye open/closed detection
- Emotion analysis in background thread
- Real-time bounding box & alerts

## ▶ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Add your images inside:
   Kishore_photos/

3. Run:
   python encoding.py

Press 'q' to quit.
